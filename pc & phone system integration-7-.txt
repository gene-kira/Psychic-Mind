⚠️ WHERE IT CAN BE IMPROVED (CONS)
1. H.264 decoding is best‑effort
OpenCV decoding from TS chunks works but isn’t a persistent decoder.

2. BLE notifications are stubbed
Needs a persistent BLE session for real streaming.

3. Mesh routing is basic
It’s functional but not a full routing protocol (no DHT, no Raft).

4. Voice recognition backend is blocking
SpeechRecognition is not truly async — Whisper streaming would fix this.

5. AI pipelines are simplified
Real YOLO/Whisper postprocessing could be more complete.

6. GUI is powerful but not polished
Could use:

Real map tiles

Better camera layout

Drag‑and‑drop panels

Node health icons

7. No encryption
All communication is plaintext.

8. No persistent storage
Swarm state resets on restart.
