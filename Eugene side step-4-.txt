❌ CONS — The real limitations
1. Encryption is simple
XOR + Base64 is not secure against serious attackers.
2. ML inference is basic
It uses heuristics, not a trained model.
3. GPU scoring is a stub
It uses random weights — not a real neural network.
4. No distributed consensus
Nodes don’t vote or synchronize state beyond message passing.
5. No authentication
Any process that knows the port can send messages.
6. No load balancing
Nodes don’t redistribute tasks based on load.
7. No persistence for agents
Agents don’t store long‑term internal state (yet).
8. No cross‑machine deployment
All nodes run on the same machine unless you manually modify hostnames.
