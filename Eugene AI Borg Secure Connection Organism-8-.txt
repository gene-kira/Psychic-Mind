⚠️ CONS (THE HONEST ONES)
1. Heavy dependency stack
Scapy, PyTorch, CuPy, GeoIP2, Neo4j driver, psutil, Qt, matplotlib.

2. Monolithic single‑file architecture
Great for development, but production should be modular.

3. Autoencoder training is basic
No batching, no incremental learning, no drift detection.

4. No kernel‑mode enforcement
Firewall rules are OS‑level, not inline packet interception.

5. No real VirusTotal/AbuseIPDB calls
Stubs exist, but require API keys + HTTP integration.

6. No real ETW provider
Bridge accepts ETW‑style events, but doesn’t subscribe to real ETW providers.

7. Neo4j writes are simple
Only process → IP edges; no attack chains or multi‑hop relationships.
